============
Installation
============


Stable release
--------------

To install ml_snek, run this command in your terminal:

.. code-block:: console

    $ pip install ml_snek

This is the preferred method to install ml_snek, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ml_snek can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/gregjohnso/ml_snek

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/gregjohnso/ml_snek/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/gregjohnso/ml_snek
.. _tarball: https://github.com/gregjohnso/ml_snek/tarball/master
